Anchors and it's uses.

In HTML we can use something called anchors, these are basically links that you can click and that will lead you to somewhere else. Anchors use the "a" tag.

Example:

click me :)

If you clicked the anchor you will notice that it will lead you to nowhere, this is because within our code we didn't add end point or a site to lead you to.

Better example:

click me :D

In this last example we still used the anchor tag, only this time we added an href attribute which we can use to add an end point and connect a link. The format is as following "href = 'link'".

Images

To insert images we use the "img" tag. Unlike other tags this specific one doesn't have a closing tag. This tag uses an attrivute really similar to what an anchor uses. We use "src" rather than "href" but they ultimately work in a similar way.
